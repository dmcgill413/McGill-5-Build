# 5-Build
# Multi-Page Indie Film Website

This project expands a starter single-page website into a complete multi-page site using semantic HTML, mobile-first CSS, and accessible design practices. The goal of this Build is to demonstrate the ability to organize files, connect pages using relative paths, and publish a working multi-page website online.

# Pages Included
index.html Home page  
screenings.html Upcoming screenings  
workshops.html Workshops and learning opportunities  
collaborations.html Collaboration and community projects
